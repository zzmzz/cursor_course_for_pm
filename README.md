# 产品经理 Cursor 上手课程

> 面向零编程基础的产品经理，教你用 AI 搭原型、写方案。

## 这门课适合谁

- 完全没写过代码的产品经理
- 想用 AI 提升工作效率的 PM
- 对 Cursor / AI 编程工具好奇，但不知道从哪开始的人

## 学完能做什么

1. **搭建可交互的产品原型** —— 不再只是画图，而是能生成真正能点、能填、能操作的网页
2. **用 AI 写产品方案文档** —— PRD、竞品分析、技术方案，AI 帮你起草和迭代
3. **看懂项目代码** —— 和研发沟通时不再一脸懵
4. **独立运行和预览项目** —— 在浏览器里看到你搭的原型

## 学习建议

- 每课 20-30 分钟，边看边操作效果最好
- 不要光看，一定要跟着做课后练习
- 遇到问题可以直接问 Cursor 的 AI，它就是你的随身教练
